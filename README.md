# AmigoConnect
[Puedes ver la página web aquí](https://amigoconnect-d7900.web.app/)

Esta es una plataforma para estudiantes de la Universidad Catolica Luis Amigó.
Permite a los estudiantes compartir sus proyectos y tener un espacio para mostrar sus habilidades y conocimientos.

## Características

El sitio permite a los usuarios iniciar sesión con sus cuentas de Google academicas.
Los usuarios pueden subir proyectos con imágenes y descripciones.
Cada usuario cuenta con su perfil e información de contacto.
Los proyectos se pueden filtrar por semestre y categoría.
Los proyectos se pueden ordenar por fecha.

## Tecnologías

- React
- JavaScript
- Vite
- Firebase
